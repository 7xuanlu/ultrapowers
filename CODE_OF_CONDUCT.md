# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1** as its code of conduct. The full,
canonical text is maintained at:

- https://www.contributor-covenant.org/version/2/1/code_of_conduct/
- Plain-text: https://www.contributor-covenant.org/version/2/1/code_of_conduct/code_of_conduct.md

By participating in this project (issues, pull requests, discussions) you agree to uphold it.

## Scope

It applies within all project spaces and in public spaces when an individual is representing the
project or its community.

## Reporting

Concerns may be reported privately to the maintainer, [@7xuanlu](https://github.com/7xuanlu).
All reports will be reviewed and handled discreetly. Maintainers are responsible for clarifying and
enforcing the standards in the Covenant and may take appropriate, fair corrective action.
